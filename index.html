HTML and CSS Cheatsheet

HTML

Basic tags

Tag Name Description
<html> </html> HyperTextMarkupLink Creates an HTML document
<head> </head> Head Sets off the title and other info that isn't displayed
<body> </body> Body Sets of the visible area of the document
<title> </title> Title Puts the name of the document in the Title bar and is the name that gets bookmarked

Body Attributes(only used in email newsletters)

Attribute Name Description
bgcolor=? Background Color Changes the background color of the element using a color name or hex value
text=? Text color Sets the text color using a color name or hex value
link=? Link color Sets the color of links using a color name or hex value
vlink=? Visited link color Sets the color of visited links using a color name or hex value
alink=? Active link color Sets the color of active links using a color name or hex value

Text Tags

Tag Name Description
<pre> </pre> Preformatted Creates preformatted text
<h1> </h1> ---> <h6> </h6> Headlines Creates headlines where h1 is the biggest and h6 is the smallest
<b> </b> Bold Creates bold text(Should use <strong> instead)
<i> </i> Italic Creates italicized text(Should use <em> instead)
<tt> </tt> Typewriter Text Creates typewriter style text
<code> </code> Source code Defines source code, usually monospace
<cite> </cite> Citation Creates a citation, usually in italics
<address> </address> Address Creates address section, usually in italics
<em> </em> Emphasize Emphasizes a word, usually in italics
<strong> </strong> Strong Emphasizes a word, usually in bold
<font size=?> </font> Font Size Sets the font size from 1 to 7(Should use css instead)
<font color=?> </font> Font Color Sets the font color(Should use css instead)
<font face=?> </font> Font Face Defines the font face(Should use css instead)

Links

Tag Name Description
<a href="url">clickable text </a> Hyperlink Creates a hyperlink to the Uniform Resource Locator(url)
<a href="mailto:EMAIL_ADDRESS">clickable text </a> Hyperlink Creates a hyperlink to an email address
< name="NAME"> Target location Creates a target location in the document
< href="#NAME"> </a> Jump to location Creates a link to the target location

Lists

Tag Name Description
<ul> </ul> Unordered List Creates an unordered list
<ol start=?> </ol> Ordered List Creates an ordered list that starts at the "start=?" number
<li> </li> List Item Creates a list item in the list
<dl> </dl> Definition List Creates a definition list
<dt> Definition Term Precedes each defintion
<dd> Definition Data Precedes each definition

Graphical elements

Tag Name Description
<hr> Horizontal Rule Inserts a horizontal rule
<hr size=?> Horizontal Rule Size Sets size(height) of horizontal rule
<hr width=?> Horizontal Rule Width Sets width of rule(as a % of pixel length)
<hr noshade> Horizontal Rule No Shade Creates a horizontal rule with no shadow
<img src="url"> Image Adds a image from the url
<img src="url" align=?> Image Align Aligns image(Use css instead)
<img src="url" border=?> Image Border Sets size of border around the image(Use css instead)
<img src="url" height=?> Image Height Sets height of image in pixels
<img src="url" width=?> Image Width Sets the width of image in pixels
<img src="url" alt=?> Image Alternate Sets the alternate text for browsers that can't process images(required by the ADA)

Tables

Tag Name Description
<table> </table> Table Creates a table
<tr> </tr> Table Row Sets of each row in a table
<td> </td> Table Data Sets of each cell in a row
<th> </th> Table Header Sets off the table header(A normal cell with bold text)

Table Attributes(Only used for email newsletters)

Tag Name Description
<table border=?> Table Border Sets width of the table borders
<table cellspacing=?> Table Cellspacing Sets amount of space between table cells
<table cellpadding=?> Table Cellpadding Sets amount of space between a cell's border and it's contents
<table width=?> Table Width Sets the width of the table in pixels or as a percentage
<tr align=?> Table Row Align Sets the alignment for cells within the row
<td align=?> Table Data Align Sets the align for cells(left/center/right)
<tr valign=?> Table Row Vertical Align Sets the vertical alignment for cells within the row
<td valign=?> Table Data Vertical Align Sets the vertical alignment for cells(top/middle/bottom)
<td rowspan=?> Table Data Rowspan Sets the number of rows a cell should span(default=1)
<td colspan=?> Table Data Column Span Sets the number of columns a cell should span
<td nowrap> Table Data No Wrap Prevents lines within a cell from being broken to fit

CSS

Font Properties

Property Description
Font-Family Changes the font family of the selected element
Font-Style Changes text(Normal/oblique/italics) of the selected element
Font-variant Used to display font in normal or small caps of the selected element
Font-Weight Used to specify the weight of the font(Bold, normal, etc.) of the selected element
Font-Size Used to modify the size of the displayed font of the selected element
Font Used to combine all properties of fonts of selected element, eg.: p{font: italic bold 12pt/14pt Times, serif;}

Color and Background Properties

Property Description
Color Changes the color of the text of the selected element
Background-Color Sets the background color of the selected element
Background-Image Sets the background image of an element
Background-Repeat Determines how a specified background image is repeated
Background-Attachment Determines if a specified background image will scroll with the content or be fixed with regard to the canvas
Background Used to combine all properties of the background, eg.: body{background: red url(leaves.jpg) no-repeat bottom right;}

Text Properties

Property Description
Word-Spacing Defines an additional amount of space between words
Letter-Spacing Defines an additional amount of space between characters
Text-Decoration Allows text to be decorated through one of five properties(underline/overline/line-through/blink/none)
Vertical-Align Used to alter the vertical positioning of an inline element
Text-Transform Allows for capitalizing or decapitalizing of letters
Text-Align Used to justify text left, center, right and justify
Text-Indent Used to specify the amount of indentation prior to the first line of text
Line-Height Used to control the spacing between baselines of text

Classification Properties

Property Description
List-Style-Type Specifies the type of list-item marker, and is used if list-style-image is none or if image loading is turned off
List-Style-Image Specifies the image that will be used as list-item marker when image loading is turned on, replacing the marker specified in the list-style-type property
List-Style_Position Determines where the marker is placed in regard to the list item

Box Properties

Property Description
Margin Sets the margins of an element by specifying the length or percentage
Padding Sets the amount of space between the borders and the content of the selector
Border-Width Sets the width of the element's border
Border-Color Sets the color of the border of the element
Border-Style Sets the style of the border of the element
Border Sets the width, style and color of the border of the element
Width Sets the width of the element
Height Sets the height of the element
Float Allows text to wrap around an element(left/right/none)
Clear Specifies whether an element allows floating elements to its sides
Top Moves the element the specified distance from the top of the original position
Bottom Moves the element the specified distance from the bottom of the original position
Left Moves the element the specified distance from the left of the original position
Right Moves the element the specified distance from the right of the original position
